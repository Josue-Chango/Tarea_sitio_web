# Tarea_sitio_web

Empieza en el index, en el cual se encuentran varias secciones para navegar entre diferentes paginas dinamicamente

En la seccion de "home" esta informacion principal de la pasteleria

En la seccion de "menu" se encuentran varios postres donde se pueden alegir con una breve descripcion

En la seccion "Sobre Nosotros" se encuentra una pequeña descripcion del restaurante y una seccion desplegable donde se encuentra un poco mas de descripcion y abajo esta un pequeño video publicitario

En la seccion "Reservacion" se encuentra casillas donde se debe llenar para reservar, ademas se encuentra la ubicacion del restaurante
